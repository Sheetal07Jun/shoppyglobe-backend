# ShoppyGlobe Backend API

## Project Overview

This project is the backend API for the ShoppyGlobe e-commerce application built using Node.js, Express.js, and MongoDB.

## Features

* User Registration and Login
* JWT Authentication
* Product APIs
* Cart APIs
* Protected Routes
* MongoDB Integration
* Error Handling Middleware

## Technologies Used

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* JWT
* bcryptjs
* ThunderClient

## API Routes

### Products

* GET /products
* GET /products/:id
* POST /products

### Cart

* POST /cart
* PUT /cart/:id
* DELETE /cart/:id

### Authentication

* POST /auth/register
* POST /auth/login

## Run Project

Install dependencies:

npm install

Run server:

npm run dev
